This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

Patient

A patient resource for an HIV Patient
Lab Results

The result of the lab test which determines whether the patient is infected with HIV or not.
HIV Recency Test

This profile is to record the HIV recency testing data for a patient.
ARV Treatment

This profile is to record prescribed ARV regimen against a given therapeutic line.
Practitioner

The healthcare professional who has been assigned to a given lab task.
HIV Care Medication Request

This profile is for recording the Patient’s ARV Dispensing quantity in days.
Lab Order Request Location

What is the location of the organization responsible for conducting the examination of the individual’s sample?
Patient Enrollment Type

This profile is used to enrol the patient into HIV care.
HIV Organization Patient Is Transferred To

Organization who is receiving the patient as being transferred in.
Absolute CD4 Count

This profile is for recording the Patient’s CD4 Count.
Date HIV Test Done

This profile is to record the date when HIV test was done for a patient.
Lab Task

Assists with tracking the state of the lab order and its completion status.
HIV Recency Test Result

This profile is to record the HIV recency test result data for a patient.
Patient Death

This profile captures the death outcome for a patient.
HIV Organization

Organization providing HIV Testing Services.
Lab Order

A service request that initiates the need for the lab to collect the test sample.
Diagnostic Report

A report as a result of the lab task being completed.
Diagnosis

This profile represents the confirmation of HIV diagnosis.
Target Facility Encounter

This profile represents the current facility at which the patient is receiving treatment.
CD4 Percentage

This profile is for recording the patient’s CD4 percentage.
Transferred Out Service Request

A service request that initiates the need for a patient to be transferred to another organization.
Transferring Facility Encounter

This profile represents the facility the patient is being transferred from.
Viral Load Specimen

The test sample that was collected for the initiated lab order.

Structures: Extension Definitions

These define constraints on FHIR data types for systems conforming to this implementation guide.

Next Appointment Date

A date representing the patient’s next scheduled appointment.
Key HIV Population

Populations who are at higher risk for HIV.
ART Regimen Switched Or Substituted

The ARV regimen has been switched to a new ARV regimen or has been substituted by another ARV regimen.
ART Regimen Line

Therapeutic lines that are used to classify the patient’s currently prescribed ARV regimen.
